
Python shutile
===========================

1.shutile 简介
---------------------------

   ``shutile`` 模块提供了一系列对文件和文件集合的高阶操作，特别是提供了一些文件拷贝和删除的函数。
   对于单个文件的操作，使用 ``os`` 模块。

   - 常用 API:

      - copyfileobj
      - copyfile
      - copymode
      - copystat
      - copy
      - copy2
      - ignore_patterns
      - copytree
      - rmtree
      - move
